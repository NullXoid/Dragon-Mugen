# Arena Mode Scope

## Included

- Arena menu entry
- Character select
- CPU count from 1 to 3
- Random CPU loading
- Free-for-all mode
- Last-fighter-standing win condition
- Winner screen
- End screen
- Return to menu

## Excluded

- Z-axis or Power Stone movement
- Scroller mode
- World maps
- NPCs
- Shops
- Equipment
- Crafting
- Story mode
- Tournament progression
- Advanced arena AI
