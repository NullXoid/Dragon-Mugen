# KFM Motif Data

This folder contains the Kung Fu Man sample motif/story support files from the 2001 DOS M.U.G.E.N package.

## Files

- `system.def` - KFM-themed system/menu definition.
- `select.def` - select list for the KFM sample setup.
- `logo.def` / `logo.sff` - logo storyboard.
- `intro.def` / `intro.sff` - intro storyboard.
- `gameover.def` / `gameover.sff` - game over storyboard.
- `credits.def` / `credits.sff` - credits storyboard.
- `kfmtitle.sff` - title sprites.

These are not part of the first training-mode milestone, but they are useful later when implementing M.U.G.E.N-style storyboards and menu flow.
