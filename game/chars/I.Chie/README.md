# I.Chie

Reserved original Dragon MUGEN advanced AI and compatibility benchmark character.

This folder intentionally contains no selectable character files yet. It exists so the future advanced test fighter is tracked in the M.U.G.E.N-style content tree instead of living only in chat or loose notes.

Do not add `I.Chie` to `game/data/select.def` until the folder contains a real M.U.G.E.N-compatible fileset:

- `I.Chie.def`
- `I.Chie.cmd`
- `I.Chie.cns`
- `I.Chie.air`
- `I.Chie.sff`

Planned purpose: advanced AI and compatibility stress tests for variables, helpers, projectiles, target controllers, superpause, explods, palette effects, and richer decision logic.
