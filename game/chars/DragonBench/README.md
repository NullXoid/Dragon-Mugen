# DragonBench

Reserved original Dragon MUGEN benchmark character.

This folder intentionally contains no selectable character files yet. It exists so the future benchmark fighter is tracked in the M.U.G.E.N-style content tree instead of living only in chat or loose notes.

Do not add `DragonBench` to `game/data/select.def` until the folder contains a real M.U.G.E.N-compatible fileset:

- `DragonBench.def`
- `DragonBench.cmd`
- `DragonBench.cns`
- `DragonBench.air`
- `DragonBench.sff`

Planned purpose: deterministic regression fighter for movement, hit, fall, guard, projectile, helper, super, and target/custom-state compatibility tests.
