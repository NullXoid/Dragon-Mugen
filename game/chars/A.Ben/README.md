# A.Ben

Reserved original Dragon MUGEN basic AI benchmark character.

This folder intentionally contains no selectable character files yet. It exists so the future AI opponent is tracked in the M.U.G.E.N-style content tree instead of living only in chat or loose notes.

Do not add `A.Ben` to `game/data/select.def` until the folder contains a real M.U.G.E.N-compatible fileset:

- `A.Ben.def`
- `A.Ben.cmd`
- `A.Ben.cns`
- `A.Ben.air`
- `A.Ben.sff`

Planned purpose: readable low-complexity AI opponent for dash, guard, spacing, normals, one special, and round-safe decision tests.
