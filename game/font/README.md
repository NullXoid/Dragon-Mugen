# Fonts

This folder is for font assets used by menus, fight UI, storyboards, debug views, and editor previews.

M.U.G.E.N used `.fnt` files and later `.def` font definitions. This engine should eventually support a clean font format while preserving enough M.U.G.E.N compatibility for reference content.

Current milestone status: reserved. Font rendering comes after SDL3/bgfx bootstrap.
