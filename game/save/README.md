# Save

Reserved for local runtime data.

Future files may include:

- user settings
- controller mappings
- profiles
- unlocks
- character progression
- tournament records
- local campaign saves

Do not place character or stage content here.

## Dragon Settings

Local Dragon-only player preferences belong here, planned as `settings.def`.

Examples:

- window/fullscreen preferences
- canvas size preference
- input bindings
- gamepad assignment and Xbox/PlayStation prompt label preference
- training-option defaults chosen by the local player
- profile/unlock/progression data

Project defaults belong in `game/data/dragon.def`; local overrides belong here. Do not write local settings into M.U.G.E.N character, stage, system, or fight files.
