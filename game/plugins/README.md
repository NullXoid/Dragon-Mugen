# Plugins

Reserved for optional runtime extensions.

Plugins are out of scope for the first playable training sandbox. This folder exists so the runtime layout has a stable place for future add-ons without disrupting the M.U.G.E.N-style content folders.

Plugins must be optional. Core KFM compatibility, stage loading, character loading, and the M.U.G.E.N folder contract must not depend on a plugin being installed.

Any plugin-facing configuration that becomes part of the project format must be documented in [../../docs/DRAGON_EXTENSIONS.md](../../docs/DRAGON_EXTENSIONS.md).
